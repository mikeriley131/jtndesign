# boilerplate-jekyll
Pyrographic Design's HTML/CSS boilerplate for builds using Jekyll

## Getting Started
1. Clone the repo to your local machine
2. `npm install`
3. `bower install susy`
4. `bower install jquery` (optional)
5. `grunt`
6. Initialize a Git repo
7. Get developing
